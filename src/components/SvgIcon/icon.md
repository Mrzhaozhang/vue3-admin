# SvgIcon svg 图标

提供上百种 svg 格式图标，简单易用，有线框和实底两种风格

## 何时使用

- 需要使用字体图标时。
- 需要可配置旋转时。

## Attributes

| 参数      | 说明                 | 类型      | 可选值 | 默认值 |
| --------- | -------------------- | --------- | ------ | ------ |
| iconClass | 图标名称，文件名即可 | `string`  | -      | -      |
| className | 自定义 class 类名    | `string`  | -      | -      |
| spin      | 自动 360deg 旋转     | `boolean` | -      | false  |
| rotate    | 旋转角度             | `number`  | -      | -      |

## Events

| name | 说明         | 回调参数 |
| ---- | ------------ | -------- |
| -    | dom 原生事件 | -        |
